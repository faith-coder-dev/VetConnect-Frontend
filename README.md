# VetConnect Frontend

A React application connecting pet owners with veterinarians. Users can browse veterinarians, view details, search, and book appointments.

## Features

- Browse and view vet details (specialty, location, contact, ratings)  
- Search vets by name, location, or specialty  
- Book appointments  
- Add new vets (admin feature)  
- Responsive design for mobile  
- Vet rating system  

## Technologies

- React 18, React Router v6  
- React Bootstrap & Bootstrap 5  
- JSON Server (mock API)  
- Express & CORS
